- Go to the Developer console at: https://console.developers.google.com
- Create your own project (you can use the same project for both demo's)
- Go to APIs & Auth, select API's and turn on the BigQuery API
- Go to Credentials and create a new Service Account, 
notice that this will download a P12 certificate file. 
You will need to include this certificate in this project.
- Now copy the Project-ID and Service account email to Main.java
- Enjoy.
